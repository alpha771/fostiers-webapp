
#Commit Message Rule

Purpose | Value
--- | ---  
Fdir| Add New File Or Directory
Feature| Add New Feature
Bug| Fixing Bug
Refactor|Refactoring
Docs|Add or Edit Docs (Like readme etc)

> `<Purpose>(<file_name or path>):message`

### Example

> `Bug(handler.go):returning json empty`
